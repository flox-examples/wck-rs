# WCK

Check for adverse weather conditions

## Usage

```
wck 0.1.0
Check for adverse weather conditions

USAGE:
    wck [LOCATION]

ARGS:
    <LOCATION>    The location to check the weather at. Leave empty to check the weather near
                  you

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information
```
